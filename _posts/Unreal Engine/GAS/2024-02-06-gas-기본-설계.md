## AttributeSet

GAS 에서 가장 기본이 되는 요소입니다.

1. 프로젝트에서 사용할 AttributeSet 클래스 작성
2. AbilitySystemComponent 에 AttributeSet 등록
3. AbilitySystemComponent 에서 AttributeSet 초기화

> AttributeSet 등록 및 초기화는 OnRegister 이벤트에서 수행하는 것이 좋을 것 같습니다.

### 1. 프로젝트에서 사용할 AttributeSet 클래스 작성

`FGameplayAttributeData` 와 제공되는 매크로를 사용하여 작성하면 됩니다.

### 2. AbilitySystemComponent 에 AttributeSet 등록

```c++
void UMyAbilitySystemComponent::CreateDefaultAttributes()
{
  for (const TSubclassOf<UAttributeSet> DefaultAttribute : DefaultAttributes)
  {
    // 유효성 검사
    if(DefaultAttribute == nullptr) continue;

    // AttributeSet 생성 및 등록
    UAttributeSet* Attribute = NewObject<UAttributeSet>(GetOwner(), DefaultAttribute);
    AddSpawnedAttribute(Attribute);
  }
}
```

### 3. AbilitySystemComponent 에서 AttributeSet 초기화

AbilitySystemComponent 에서 AttributeSet 을 초기화하는 방법은 기본적으로 데이터 테이블을 활용하는 방법입니다.
그러나 개인적으로 불편한 점도 많았고 관련 주석에서도 아직 지원이 부족하기 때문에 GameplayEffect 를 활용하는 것도 방법이라는 조언을 보았습니다.

```c++
void UMyAbilitySystemComponent::ApplyDefaultEffects()
{
  for (const TSubclassOf<UGameplayEffect> DefaultEffect : DefaultEffects)
  {
    FGameplayEffectSpecHandle GameplayEffectSpecHandle = MakeOutgoingSpec(DefaultEffect, 1, MakeEffectContext());
    if(GameplayEffectSpecHandle.IsValid())
    {
      ApplyGameplayEffectSpecToSelf(*GameplayEffectSpecHandle.Data.Get());
    }
  }
}
```
